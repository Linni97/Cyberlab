# CyberLab

🔗 **Live site:** [https://nocryptoshade.github.io/cryptoshade/](https://nocryptoshade.github.io/cryptoshade/)

## About

CyberLab is a free, browser-based platform for learning cybersecurity through hands-on practice. The idea is simple: security concepts stick better when you actually do something, not just read about it. Every topic comes with interactive labs, a built-in terminal simulator, and tasks at three difficulty levels — no subscriptions, no installations, no accounts required.

## What's inside

The platform is built around seven modules, each covering a core area of cybersecurity:

| Module | Topic | Tasks |
|--------|-------|-------|
| 1 | Network commands — ping, ipconfig, nslookup, netstat, ssh, traceroute | 6 |
| 2 | Cryptography — symmetric/asymmetric encryption, hashing, salt, TLS | 6 |
| 3 | XSS & Web security — reflected, stored, DOM-based XSS, CSP | 7 |
| 4 | IDS, IPS & Defense in Depth — Snort, SIEM, alert fatigue | 6 |
| 5 | Email protocols — SMTP, IMAP, SPF, DKIM, DMARC | 5 |
| 6 | Kali Linux & Pentesting — nmap, Gobuster, Hydra, John, Netcat + CTF | 7 |
| 7 | Web Vulnerabilities — HTML/CSS, CIA triad, SQL Injection, XSS, CSP builder | 11 |

Each module has tasks tagged easy, medium, or hard, with hints and solutions available on demand. Progress is saved locally in the browser.
